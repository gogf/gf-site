  • 框架升级-up
版本：2.9.x(Latest)
提示

该命令从框架 v2.3 版本开始提供。

使用方式​

$ gf up -h
USAGE
    gf up [OPTION]

OPTION
    -a, --all    upgrade both version and cli, auto fix codes
    -c, --cli    also upgrade CLI tool
    -f, --fix    auto fix codes(it only make sense if cli is to be upgraded)
    -h, --help   more information about this command

EXAMPLE
    gf up
    gf up -a
    gf up -c
    gf up -cf

用以实现版本更新，同时更新主框架版本、社区组件版本到最新版本。

选项说明：

名称含义
all同时更新 cli 工具版本，并且自动修复本地代码在升级中的不兼容变更
fix升级时自动修复本地代码在升级中的不兼容变更
cli升级时同时更新 cli 工具版本
