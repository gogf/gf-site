Go 1.10 版本于 2018 年 2 月发布，主要引入了构建缓存和测试缓存。

## 主要变化

### 工具链

- **构建缓存**: `go build` 现在会缓存编译后的包，显著加快了增量构建的速度。
- **测试缓存**: `go test` 现在也会缓存成功的测试结果。如果代码没有变更，再次运行测试会直接输出缓存的结果（标记为 `(cached)`）。

### 语言变化

- **无变化**: Go 1.10 没有语言规范的变更。

### 标准库

- **`strings.Builder`**: 新增 `strings.Builder` 类型，用于高效地构建字符串，类似于 `bytes.Buffer` 但针对字符串优化。
- **`math/rand`**: `math/rand` 包现在使用更快的算法。

### 运行时

- **CGO**: 允许在 C 代码和 Go 代码之间直接传递 Go 字符串（在某些限制下）。
