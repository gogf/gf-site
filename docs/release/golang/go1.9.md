Go 1.9 版本于 2017 年 8 月发布，引入了类型别名和并行编译。

## 主要变化

### 语言变化

- **类型别名 (Type Aliases)**: 引入了类型别名语法，用于支持大规模代码重构（例如将类型从一个包移动到另一个包）。

    ```go
    type T1 = T2
    ```

### 工具链

- **并行编译**: 编译器现在支持并行编译包中的函数，利用多核 CPU 加快编译速度。
- **`./...`**: `go test ./...` 现在会匹配当前目录及所有子目录下的包（不再排除 vendor 目录，除非显式排除）。

### 标准库

- **`math/bits`**: 新增 `math/bits` 包，提供位操作函数。
- **`sync.Map`**: 新增 `sync.Map` 类型，提供并发安全的 Map 实现，适用于读多写少的场景。
- **`time`**: `time.Time` 现在单调递增，即使系统时间被修改，计算时间差也是准确的。
