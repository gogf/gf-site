设计思想是 `GoFrame` 框架的灵魂，同时对于使用者来讲，是不可或缺的内功心法。

授人予鱼不如授人以渔，比起技巧性的框架、组件使用，我们想尽可能地告诉大家为什么这样、为什么那样。

`GoFrame` 有其独特的设计思想，理解了 `GoFrame` 的设计思想，您就理解了 `GoFrame` 的全部。
