虽然 `gvalid` 已经内置了常见的数十种校验规则，但是在部分业务场景下我们需要自定义校验规则，特别是一些可以重复使用的业务相关的校验规则。当然， `gvalid` 如此的强大和贴心，她已经为您考虑得如此周全。自定义校验规则可以实现灵活性强，可复用性高的校验特性。
