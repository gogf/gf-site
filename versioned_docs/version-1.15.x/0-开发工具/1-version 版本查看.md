使用方式：

- `gf -v`
- `gf version`

用以查看当前 `gf` 命令行工具编译时的版本信息。例如：

```bash
$ gf -v
GoFrame CLI Tool v0.7.5, https://goframe.org
Built Detail:
  Go Version:  go1.13.5
  GF Version:  v1.13.0
  Git Commit:  0ab2a4a9e753f4f5fae1e96ecf40cae94895d494
  Built Time:  2020-06-05 23:30:39
```
