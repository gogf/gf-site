## 基本介绍

`GoFrame` 框架支持全自动化的接口文档生成，保障代码与接口文档同步维护，自动生成的接口文档采用的是标准的 `OpenAPIv3` 协议。在介绍 `OpenAPIv3` 协议之前，建议您先阅读了解框架的规范路由特性： [路由注册-规范路由](../路由管理/路由管理-路由注册/路由注册-规范路由/路由注册-规范路由.md) 。
