## 基本介绍

`GoFrame` 框架提供了常用的 `I18N` 国际化组件，由 `gi18n` 模块实现。

**使用方式：**

```go
import "github.com/gogf/gf/v2/i18n/gi18n"
```

**接口文档**：

[https://pkg.go.dev/github.com/gogf/gf/v2/i18n/gi18n](https://pkg.go.dev/github.com/gogf/gf/v2/i18n/gi18n)
